AURA - Server Cloner
📌 Описание
AURA — это профессиональный инструмент для полного клонирования структуры Discord серверов через браузер. Скрипт позволяет быстро и эффективно копировать всю структуру сервера: роли, каналы, категории, права доступа и настройки.

🚀 Возможности
Что клонируется:
✅ Название сервера

✅ Баннер сервера

✅ Все роли с полными правами (цвет, положение, иконки)

✅ Категории и каналы (текстовые, голосовые, новостные)

✅ Права доступа на каналы (роли и пользователи)

✅ Настройки сервера (системные каналы, AFK, уровень верификации и др.)

✅ Приветственный экран

Что НЕ клонируется:
❌ Аватарка сервера

❌ Сообщения в каналах

❌ Участники сервера

❌ Вебхуки

❌ Интеграции

🛠 Технические характеристики
Параметр	Значение
Язык	JavaScript (ES6+)
API	Discord API v10
Хранение токена	sessionStorage (только локально)
Требования	Браузер с поддержкой Fetch API
Тип	Клиентское приложение (SPA)
🔐 Безопасность
✅ Токен хранится только в sessionStorage браузера

✅ Токен удаляется при закрытии вкладки

✅ Токен НЕ отправляется на сторонние сервера

✅ Все запросы идут только на официальный Discord API

✅ Нет серверной части — весь код работает в браузере

✅ Открытый исходный код — можно проверить каждую строку

📋 Требования
Токен Discord (аккаунт с правами администратора на целевом сервере)

ID исходного сервера (откуда клонируем)

ID целевого сервера (куда клонируем)

Права администратора на целевом сервере

🔧 Как использовать
Войдите в систему, вставив Discord токен

Введите ID исходного и целевого серверов

Нажмите "Проверить" для валидации серверов

Нажмите "Начать клонирование"

Подтвердите правила использования

Ожидайте завершения процесса (прогресс отображается)
