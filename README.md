## 📌 О проекте

**AURA** — это мощный и удобный инструмент для полного клонирования Discord серверов. Всё работает прямо в вашем браузере, не требует установки или дополнительного ПО. Просто откройте сайт, вставьте токен и клонируйте сервер за несколько минут.

### ✨ Особенности

- 🎯 **Полное клонирование** — роли, каналы, категории, права доступа
- 🚀 **Быстрота** — оптимизированные запросы к Discord API
- 🔒 **Безопасность** — токен хранится только локально в вашем браузере
- 🎨 **Современный интерфейс** — тёмная/светлая тема, анимации
- 📊 **Детальное логирование** — каждый шаг процесса отображается

## 🛠 Возможности

### Что клонируется:

| Компонент | Статус |
|-----------|--------|
| Название сервера | ✅ |
| Баннер сервера | ✅ |
| Все роли с правами | ✅ |
| Категории и каналы | ✅ |
| Права доступа | ✅ |
| Настройки сервера | ✅ |
| Приветственный экран | ✅ |

### Что НЕ клонируется:

| Компонент | Причина |
|-----------|---------|
| Аватарка сервера | Технические ограничения API |
| Сообщения | Не предусмотрено |
| Участники | Не предусмотрено |
| Вебхуки | Не предусмотрено |

## 🚀 Быстрый старт

1. **Откройте сайт**
2. **Вставьте Discord токен** и войдите
3. **Укажите ID исходного и целевого серверов**
4. **Нажмите "Начать клонирование"**
5. **Подтвердите правила использования**
6. **Наблюдайте за прогрессом** в реальном времени

## 🔒 Безопасность

- ✅ Токен хранится **ТОЛЬКО** в sessionStorage вашего браузера
- ✅ Токен **удаляется** при закрытии вкладки
- ✅ Токен **НЕ отправляется** на сторонние сервера
- ✅ Все запросы идут **ТОЛЬКО** на официальный Discord API
- ✅ Нет серверной части — полная прозрачность
